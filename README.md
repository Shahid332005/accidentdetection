This is a computer vision model working in realtime using CCTV installed by NHAI on the various places on Indian roads. 
This model will detect realtime accident using cameras and notify the authorities regarding the same.
A message will be sent to the official phones of respective authorities to provide help as soon as possible.

The future scope of this project is to build a automated system using self driving ambulances and camera sensors working collaboratively
in order to provide help to the affected people without any delay and engagement of real human effort in understanding and acting the situation.
